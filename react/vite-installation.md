1. make sure you have node.js and vscode installed

```bash
node -v
npm -v
```

2. Time to make our new directory

```bash
npm create vite@latest my-vite-app
```

fill the details asked

3. Time to go to that directory

```bash
cd my-vite-app
```

4. Install node modules

```bash
npm install
```

5. run it

```bash
npm run dev
```
